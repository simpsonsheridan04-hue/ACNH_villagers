This project looks at the villagers in the game Animal Crossing New Horizons (ACNH), created by Nintendo. This project will analyze the different characterisitics and traits between villagers and reflect on the differences between species types and genders. 

## Resources used: 
* Python
  * libraries: pandas, matplotlib.pyplot, seaborn, numpy
  * methods, visuals, loops, and functions
 
## Background Information: 

(from https://en.wikipedia.org/wiki/Animal_Crossing:_New_Horizons)
<br/><br/> 
"Animal Crossing: New Horizons is a 2020 social simulation game developed and published by Nintendo for the Nintendo Switch. It is the fifth main installment in the Animal Crossing series, following Animal Crossing: New Leaf. In New Horizons, the player controls a character who moves to a deserted island after purchasing a getaway package from Tom Nook, accomplishes assigned tasks, and develops the island as they choose. They can gather and craft items, customize the island, and develop it into a community of anthropomorphic animals." 

## Data Source: 

This dataset was gathered from Kaggle (https://www.kaggle.com/datasets/jessicali9530/animal-crossing-new-horizons-nookplaza-dataset?select=villagers.csv). Please note, this does not include content of any new villagers after the initial game launch in 2020. Three updates were launched on 3/2021, 11/2021, and 1/2026, which added additonal villagers through amiibos. (source: https://www.ign.com/wikis/animal-crossing-new-horizons/Villager_List:_All_Villagers_and_Special_Characters)
